EasyVisa - Visa Approval Prediction

Ensemble Machine Learning | UT Austin PG Data Science

Business Context

EasyVisa is a consultancy helping employers obtain work visas for foreign employees. The US visa application process is complex, with OFLC (Office of Foreign Labor Certification) reviewing applications based on multiple factors. Understanding these factors helps improve approval rates and reduce processing time.

Objective

Build a classification model to predict visa certification outcomes and identify key factors influencing decisions:

  • Predict whether a visa application will be Certified or Denied
  • Identify most important features affecting certification
  • Provide actionable recommendations for improving approval rates
  • Help employers prepare stronger applications

Technical Approach

1. Data Analysis:

  • Analyzed 25,480 visa applications with employer and employee attributes
  • Explored features: education level, experience, wage, region, job category
  • Identified class imbalance (66% Certified, 34% Denied)

2. Feature Engineering:

  • Created wage ratio feature (offered vs. prevailing wage)
  • Encoded education levels with ordinal mapping
  • Applied SMOTE to handle class imbalance

3. Model Development:

  • Implemented Decision Tree, Random Forest, AdaBoost, Gradient Boosting, XGBoost
  • Performed extensive hyperparameter tuning using GridSearchCV
  • Used stratified K-fold cross-validation

Key Feature Importance

1. Prevailing Wage - Most influential factor (higher wage = higher approval)

2. Education Level - Master's/Doctorate significantly improve chances

3. Job Experience - Required experience correlates with certification

4. Region of Employment - Certain states have higher approval rates

Model Performance

Model Accuracy AUC-ROC Precision Recall
Decision Tree 81% 0.85 79% 78%
Random Forest 86% 0.90 84% 83%
Gradient Boosting 87% 0.91 85% 84%
XGBoost (Tuned) 89% 0.92 87% 86%

Business Recommendations

  • Wage positioning: Offer wages above prevailing wage threshold
  • Education emphasis: Prioritize candidates with advanced degrees
  • Experience documentation: Clearly document required job experience
  • Regional strategy: Consider employment regions with higher approval rates
