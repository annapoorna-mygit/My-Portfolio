Hotel Booking Cancellation Prediction

Machine Learning Classification Project

Project Overview

Developed a classification model to predict hotel booking cancellations, helping hotels optimize their booking strategies and reduce revenue loss from cancellations. The project involves analyzing customer behavior patterns and building predictive models.

Problem Statement

Hotel cancellations pose a significant challenge to revenue management. This project aims to predict which bookings are likely to be cancelled, enabling hotels to:

  • Implement dynamic pricing strategies
  • Optimize overbooking policies
  • Improve resource allocation and staffing
  • Enhance customer retention strategies

Methodology

Data Preprocessing:

  • Handled missing values and outliers in booking data
  • Feature engineering: created lead time categories, seasonal features
  • Encoded categorical variables (market segment, distribution channel)
  • Balanced the dataset using SMOTE for minority class

Model Selection:

  • Tested Logistic Regression, Decision Trees, Random Forest, and XGBoost
  • Performed hyperparameter tuning using RandomizedSearchCV
  • Compared models based on precision, recall, and AUC-ROC

Results

The Random Forest classifier achieved the best performance:

  • Accuracy: 87%
  • Precision: 85%
  • Recall: 83%
  • F1-Score: 84%
  • AUC-ROC: 0.91

Key Predictors: Lead time, previous cancellations, deposit type, and average daily rate were identified as the most important features.
