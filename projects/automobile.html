Automobile Performance Analysis

Comprehensive Data Analysis of Auto-MPG Dataset

Project Overview

Comprehensive analysis of automobile performance metrics focusing on fuel efficiency, horsepower, and other key factors. This project explores relationships between various vehicle attributes and their impact on miles per gallon (MPG).

Analysis Performed

  • Exploratory Data Analysis (EDA) of 398 automobile records
  • Statistical analysis of fuel efficiency trends across decades
  • Correlation analysis between vehicle weight, horsepower, and MPG
  • Visualization of performance metrics and trends

Key Findings

  • Strong negative correlation between vehicle weight and fuel efficiency
  • Significant improvement in MPG over model years (1970-1982)
  • European and Japanese cars showed better fuel efficiency than US cars
  • 4-cylinder engines demonstrated superior MPG compared to 8-cylinder engines

Technologies Used

Python Pandas NumPy Matplotlib Seaborn Jupyter Notebook
